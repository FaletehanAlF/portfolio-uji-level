Saya menggunakan pendekatan sistematis dalam setiap project:

- Mengidentifikasi kebutuhan user
- Menentukan fitur utama

### 2. Perancangan
- Mendesain struktur folder
- Membuat wireframe sederhana

- Pengujian tampilan responsive

- Debugging error

### 5. Evaluasi
- Menyederhanakan kode
- Meningkatkan UI
- Mempersiapkan pengembangan lanjutan
